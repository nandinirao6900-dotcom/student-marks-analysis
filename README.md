# Student Marks Analysis

## 📌 Overview
This project analyzes student marks using Python and Jupyter Notebook. It calculates total marks, percentages, grades, and ranks, and visualizes the data using bar charts and pie charts.

## 🚀 Features
- Store student marks
- Calculate total marks
- Calculate percentage
- Assign grades
- Rank students
- Visualize performance using charts

## 🛠️ Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Jupyter Notebook

## 📊 Visualizations
- Bar Chart
- Pie Chart

## 📁 Dataset
The project uses a sample dataset containing:
- Student Names
- Maths Marks
- Science Marks
- English Marks

## 📈 Analysis Performed
- Total marks calculation
- Percentage calculation
- Grade assignment
- Student ranking
- Subject-wise performance analysis

## ▶️ How to Run
1. Clone this repository.
2. Install the required libraries:
   ```bash
   pip install numpy pandas matplotlib
   ```
3. Open the notebook:
   ```
   Student Marks Analysis.ipynb
   ```
4. Run all cells.

## 🎯 Learning Objectives
This project demonstrates:
- Data manipulation using Pandas
- Numerical operations using NumPy
- Data visualization using Matplotlib
- Basic data analysis techniques
- Working with Jupyter Notebook
